# @mistcash/config

MIST SDK config

## Installation

```sh
pnpm add @mistcash/config
```

## Usage

```js
import { CHAMBER_ABI, CHAMBER_ADDR_MAINNET, ChamberTypedContract } from '@mistcash/config';

// Do something with the ABIs and other config data
```
